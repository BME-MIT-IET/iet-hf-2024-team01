# Drukmakori-Sivatag
Cső színei:

nincs állapot: szürke
van víz simán: kék
lyukas: világosabb piros
van víz + lyukas: sötét piros
csúszós: világos zöld
ragadós: sárga
van víz + csúszós : sötét zöld
van víz + ragadós : narancs
lyukas + csúszós: barna
lyukas + ragadós: lila
van víz + lyukas + csúszós : rózsaszín
van víz + lyukas + ragadós: fekete

Pumpa szinei:
működik: szürke
nem működik: piros 

Billentyűk:
Q: Mozgás

W: Pumpa átállítás.

E: Javítás

R: Pumpa felvétel

T: Pumpa lerakás

Z: Cső felvétel

U: Cső lerakás

I: Cső lyukasztás

O: Csúszóssá tevés

P: Ragadóssá tevés  

A: Kör vége
